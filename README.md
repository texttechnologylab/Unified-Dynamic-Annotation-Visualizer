# Unified Dynamic Annotation Visualizer

UDAV is designed to enable different disciplines
to display their automatic pre-processing results
in a schema-based and reproducible,
dynamic and interactive way without the need to
hard-code manual and user-defined visualizations
for each new project.

## Getting Started

> [!TIP]
> Please consult the [documentation](https://texttechnologylab.github.io/Unified-Dynamic-Annotation-Visualizer/) page for a more detailled and customizable setup documentation.

### Requirements

- Java version 21 or higher

### Usage

1. Clone the repository:

```
git clone https://github.com/texttechnologylab/Unified-Dynamic-Annotation-Visualizer.git
```

2. Start the `App.java` file

### Development

For setting up UDAV in an development environment, refer to our [documentation](https://texttechnologylab.github.io/Unified-Dynamic-Annotation-Visualizer/).
